# Heroicons (third-party — temporary use)

이 폴더의 SVG 는 [Heroicons](https://heroicons.com) (16x16 micro / solid) 의 일부를 임시로 가져와 SKT 자체 SVG 가 준비될 때까지 placeholder 로 사용합니다.

**라이선스**: MIT — Copyright (c) Tailwind Labs, Inc.  
원본: https://github.com/tailwindlabs/heroicons

## 사용 중인 파일

| 파일 | 매핑된 atom |
|---|---|
| qr-code.svg | atom/icon/qr-scan |
| signal.svg | atom/icon/signal |
| wifi.svg | atom/icon/wifi |
| battery-100.svg | atom/icon/battery |
| arrow-left.svg | atom/icon/arrow-left |
| share.svg | atom/icon/share |
| heart.svg | atom/icon/heart |
| gift.svg | atom/icon/gift |
| x-mark.svg | atom/icon/close |

SKT 자체 아이콘이 받아지면 `mockup/figma-icons/Normal/` (또는 적절한 카테고리) 로 교체 + 본 폴더 삭제 + `mockup/scripter/icon-registry.json` 갱신.
